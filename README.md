I'm currently working as **Community Growth Contributor** for **[Watchflow](https://github.com/warestack/watchflow)** — an open-source **Agentic GitHub Guardrails** tool.

Watchflow brings context-aware governance to repositories. It enforces smart policies (CODEOWNERS, PR approvals, linked issues, size limits, title patterns, etc.) using a simple `.watchflow/rules.yaml` file.

I'm actively helping active open-source projects adopt Watchflow by proposing clean integrations via PRs.
